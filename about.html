the Journey . . .

Background: The concept of this app was based on Netflix, Amazon Prime, or any other movie applications out there.

the Outcome expected:

  • registration for a new user
  • the user personalized profile is created
  • update the user's personal information
  • the user is able to pick and delete their favorite movies
  • login for all signed-up users
  • the ability to see all the movies

In order to create an application that can do all those, there are 3 important parts that play major roles.   please bear with me..

setting up the backEnd

Initiate the backEnd project by making a root directory

Once the root directory was generated, I switch my environment to the particular root directory I had to work on

Then, I created a package.json by running this command: $ npm init -y

Add some dependencies to the app. There were at least 4 major dependecies that I used for this app:

express : body-parser : cors : mongoose

setting up the Database

Install and Setup MongoDB

  • Create a New MongoDB Database
  • Connect to MongoDB by using Mongoose
  • Create a Mongoose Schema
  • Implementing the Server Endpoints
  • Testing the Server API with Postman
setting up with the frontEnd

I started this segment by installing Node.js as I needed to install React dependencies: $ npm install

Next, input npx create-react-app into the integrated terminal. This command will construct a client application with the name client in the current root directory.

Conclusion

I tested the app locally using Postman and http://localhost:3000.

I pushed all the codes to github and deploy this app on netlify (link coming soon). Thank you for reading. Cheers!! Happy Coding !!

What can be better

honeypotflixplay actually has a lot of room for improvement.

Things can be added such as the actors' names and their bios. Also, it would have been nice to add the live streaming to the picture or voice integration to search and/or play the chosen movie.

That being said, I am sure if I was given more time, the app would have looked better. However, all that being said, I am grateful that it came to the end on time though I struggle to comprehend each component and its relation to one another. Hopefully, I can take this app to the next level in 6 months time.
